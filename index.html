    Google Calendar and Google Tasks integration is now available: connect your Google account and synchronize one-time tasks. Task Aura remains server-less, so authorization data is stored locally on your device.

    Published on: 2026-05-06

    Update to version 1.4.0

    Progress section: - Added a new "Progress" tab to replace "Achievements". - Added a progress hub with calendar list and a built-in non-removable "Medals by day" calendar. - Added quick creation of custom calendars with tracker type selection: completed tasks or time.

    Published on: 2026-05-01

    Update to version 1.3.15

    Achievements and notes: - Added longest medal streak to achievement statistics. - Added INFO button in achievements with guidance on how to earn medals, cups, and lightning. - Reworked notes with a second compact card mode showing note title and content preview. - Added note settings: background color, tags, tag filtering in compact mode, and a flag to hide note in compact mode.

    Published on: 2026-04-21

    Update to version 1.3.14

    Calendar, tasks, and data transfer: - Updated selected-day highlighting in the weekly calendar: it stays blue by default, and turns red when a red lightning (ruby day) is active. - Fixed child-task goal counting: it now shows only child tasks relevant to the selected day. - Added a settings block for full app data/settings JSON import and export. Both file-based and text-field flows are supported. - Added mandatory confirmation before import, because current data is fully replaced.

    Published on: 2026-04-17

    Update to version 1.3.13

    Tasks and calendar: - Fixed a bug with completing tasks on past dates: now a task can be completed in the past if it has not already been completed in the future. - Added markers for days with cancelled tasks: a red exclamation mark appears in the top-right corner of the day. - Added a lightning badge in the top-left corner of the selected day. It appears only when the current day has platinum or ruby rating, all previous 7 days (excluding current) also have platinum or ruby, ruby must be 5 to 7 out of 7 days, and cancelled-task days in that 7-day window are no more than two.

    Published on: 2026-04-16

    Update to version 1.3.12

    Day goals and medals: - Medal/cup calculation now includes all regular tasks (daily, weekly, monthly), not only daily tasks. - Added "Cancelled" state for regular tasks on a selected day: cancelled tasks are not completed, but excluded from medal calculations. Routine: - Added "Pre-complete regular task" button with a picker of regular tasks for the selected day. - Selected task is instantly added as completed: no goal => completed, with goal => 100% goal progress. - If pre-completion is fully removed, the task entry disappears from the selected day.

    Published on: 2026-04-12

    Update to version 1.3.11

    Achievements: - Added a dedicated statistics panel and a separate achievements panel. - Added a full achievement list with level and progress scales, including unlimited levels. - Added creating, editing, deleting achievements, and linking multiple tasks to one achievement. Task links: - When linking a task, the app calculates its accumulated execution stats and offers adding that amount to achievement progress. - For regular tasks without goals, completed occurrences are counted. - For goal tasks, values are summed across days; for time goals, achievement progress is tracked in seconds. - After linking, each new task completion automatically increases achievement progress. - Unlinking a task does not subtract previously gained progress. - Rolling back completion or progress of a linked task now correctly decreases achievement progress. Other: - Reduced the minimum clamp font size in the Achievements tab so labels fit better on smaller phones. - Renamed the "Ultra-shining cup" reward to "Ruby cup". - Fixed day-by-day progress display for one-time goal tasks: values are now taken from the selected date state. - Added a warning when changing a past value if newer goal-progress records already exist in future dates. - Added a visual badge for tasks whose goal progress was updated today. - Fixed sorting in the one-time tasks area: drag-and-drop is now constrained to regularTasksZone. - Day goal progress for goal tasks is now calculated as the change made during the selected day (day delta), not from the moment of adding to Day goals. - One-time tasks without goals no longer appear as completed on dates before their actual completion date. - Changing completion status for one-time tasks without goals is now blocked on past dates. - Added a toggle button to auto-hide/show completed tasks in the list.

    Published on: 2026-04-10

    Update to version 1.3.10

    Navigation: - The bottom panel was redesigned for better mobile navigation and now includes the Achievements tab. - Main tabs are now Routine, Tasks, Achievements, and Notes. - News and Settings were moved into a compact More menu to keep the bar uncluttered. Day goals: - Added a Day Goals area in the Tasks section with a highlighted frame at the top. - One-time tasks can be moved to Day Goals, where progress is tracked separately for the selected day. - For count/time goals, the app asks for a daily target value when adding to Day Goals. - If Day Goals are completed and all routine tasks are done, a Ruby cup is awarded. - Without Day Goals for a date, the Ruby cup cannot be received.

    Published on: 2026-04-09

    Update to version 1.3.9.1

    - Improved task sorting in task tabs: execution time now has a higher priority than manual moves. - Tasks are sorted by time from earlier to later. - For tasks without a specified time, the nearest timed task above is used as an anchor and saved for stable sorting. - Drag-and-drop blocks impossible manual orders that contradict time-priority sorting.

    Published on: 2026-04-09
