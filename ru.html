    Последние обновления

    Опубликовано: 2026-04-30

    Обновление до версии 1.3.15

    Достижения и заметки: - В раздел статистики достижений добавлена самая длинная серия дней с медалью. - Добавлена кнопка INFO в разделе достижений с подсказкой о том, как получать медали, кубки и молнию. - Переработан раздел заметок: добавлен второй (компактный) режим отображения карточками, где видно название и часть содержимого заметки. - Для заметок добавлены новые настройки: цвет фона, теги, фильтрация по тегам в компактном режиме и флаг скрытия заметки из компактного режима.

    Опубликовано: 2026-04-21

    Обновление до версии 1.3.14

    Календарь, задачи и перенос данных: - Исправлен подсчёт для цели «Выполнение дочерних задач»: отображается количество дочерних задач только за выбранный день. - В настройки добавлен блок импорта/экспорта всех данных и настроек приложения в формате JSON. Поддержаны импорт/экспорт через файл и через текстовое поле. - Добавлено обязательное подтверждение перед импортом данных, так как текущие данные полностью заменяются.

    Опубликовано: 2026-04-17

    Обновление до версии 1.3.13

    Задачи и календарь: - Исправлен баг выполнения задач в прошлых датах: теперь можно выполнить задачу в прошлом дне, если она ещё не отмечена как выполненная в будущем. - Добавлено обозначение дней с отменой задач: в правом верхнем углу дня появляется красный восклицательный знак. - Добавлен значок молнии в левом верхнем углу выбранного дня. Он показывается только когда текущий день завершён на платину или рубин, все 7 предыдущих дней (без текущего) тоже имеют платину или рубин, среди них рубиновых дней от 5 до 7, а дней с отменой задач за этот период не больше одного.

    Опубликовано: 2026-04-16

    Обновление до версии 1.3.12

    Задания дня и медали: - Исправлен расчёт медалей и кубков: теперь учитываются все регулярные задачи (ежедневные, еженедельные, ежемесячные), а не только ежедневные. - Добавлен статус «Отменено» для регулярной задачи на выбранный день: отменённые задачи не считаются выполненными, но исключаются из расчёта медалей. Рутина: - Добавлена кнопка «Предвыполнить регулярную задачу» с выбором регулярной задачи для выбранного дня. - Выбранная задача сразу добавляется как выполненная: без цели — как завершённая, с целью — с полным прогрессом. - Если предвыполненную задачу полностью снять с выполнения, запись для выбранного дня пропадает.

    Опубликовано: 2026-04-12

    Обновление до версии 1.3.11

    Достижения: - Добавлен полноценный список достижений со шкалой прогресса и уровнями, включая бесконечные уровни. Можно создавать, редактировать, удалять достижений и привязывать задачи к достижению для автоматической работы шкалы прогресса. Привязка задач к достижениям: - При привязке задачи приложение рассчитывает накопленную статистику выполнения и предлагает добавить это число в прогресс достижения. - Для регулярных задач без цели считается количество выполнений. - Для задач с целью суммируются значения за все дни; для целей по времени прогресс достижения ведётся в секундах/минутах/часах. - При отвязке задачи ранее добавленный прогресс не уменьшается. Прочее: - Уменьшен минимальный размер шрифта во вкладке «Достижения», чтобы подписи лучше помещались на небольших экранах. - Награда «Ультра-сияющий кубок» переименована в «Рубиновый кубок». - Для обычных задач с целями исправлено отображение прогресса по дням: теперь значение берётся из состояния на выбранную дату. - Добавлено предупреждение при попытке изменить значение в прошлом, если в будущих датах уже есть более новые записи прогресса. - Добавлен визуальный ярлык для задач, у которых прогресс цели был изменён сегодня. - Для обычных задач без цели выполненность теперь не показывается в датах до дня фактического выполнения. - Запрещено менять статус выполнения обычной задачи без цели в прошлых днях. - Добавлена кнопка автоматического скрытия/показа выполненных задач в списке. - В настройках добавлен выбор первого дня недели (воскресенье или понедельник) для верхнего календарного меню. - Добавлена цель «Выполнение дочерних задач» с опцией автозавершения и группировкой дочерних задач под родительской в списке.

    Опубликовано: 2026-04-10

    Обновление до версии 1.3.10

    Навигация: - Нижняя панель переработана для удобной мобильной навигации: добавлена вкладка «Достижения». - Основные вкладки теперь: Рутина, Задачи, Достижения и Заметки. - Новости и Настройки перенесены в компактное меню «Ещё», чтобы не перегружать экран. Задания дня: - Добавлен блок «Задания дня» в разделе «Задачи» в специальной рамке сверху. - В «Задания дня» можно переносить нерегулярные задачи; для выбранного дня ведётся отдельное отслеживание прогресса. - Для целей типа «Количество» и «Время» при добавлении запрашивается дневная цель. - Если цели дня выполнены и одновременно закрыты все рутинные задачи, выдается рубиновый кубок. - Если для даты нет заданий дня, рубиновый кубок получить нельзя.

    Опубликовано: 2026-04-09

    Обновление до версии 1.3.9.1

    - Улучшена сортировка задач во вкладках: время выполнения теперь приоритетнее ручных перестановок. - Задачи сортируются по времени от более раннего к более позднему. - Для задач без указанного времени используется ближайшая задача со временем сверху как якорь, и это время сохраняется для стабильной сортировки. - В drag-and-drop заблокированы невозможные ручные перестановки, которые противоречат приоритету сортировки по времени.

    Опубликовано: 2026-04-09
